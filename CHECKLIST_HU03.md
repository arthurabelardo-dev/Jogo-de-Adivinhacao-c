# Checklist de Validacao - HU-03 (Receber pista tematica)

## Regras gerais (todos os casos)

- [ ] Cada caso possui banco proprio de pistas (sem reutilizar texto de outro caso).
- [ ] Cada caso possui no minimo 6 pistas.
- [ ] Cada caso possui no minimo 2 pistas diretas, 2 narrativas e 2 falsas.
- [ ] As pistas usam vocabulario coerente com a ambientacao do caso.
- [ ] Nenhuma pista verdadeira revela o numero secreto diretamente.
- [ ] Nenhuma pista de escopo explicito e usada (ex: "entre X e Y").
- [ ] Pistas falsas nao apontam para o numero correto.
- [ ] Durante a partida, somente pistas do caso atual podem aparecer.

## Caso 1 - O Ultimo Suspiro do Magnata

- [ ] Linguagem de mansao/cofre/familia/seguranca privada.
- [ ] Pistas narrativas combinam com investigacao patrimonial.
- [ ] Nao aparece vocabulario de cassino ou ciberataque.

## Caso 2 - Frequencia de Fuga no Cassino

- [ ] Linguagem de cassino/sistemas internos/monitoramento.
- [ ] Pistas narrativas combinam com sabotagem e seguranca do cassino.
- [ ] Nao aparece vocabulario de mansao ou data center critico.

## Caso 3 - Protocolo Apocalipse

- [ ] Linguagem de ciberseguranca/logs/firewall/trafego de rede.
- [ ] Pistas narrativas combinam com ataque digital e pericia tecnica.
- [ ] Nao aparece vocabulario de mansao ou operacao de cassino.
